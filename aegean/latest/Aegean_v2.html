AEGEAN

What is Aegean?

Aegean is a website designed to help you save time by automating simple "everyday" tasks.


How do I use Aegean?

Each action Aegean can automate is orginized into boxes. all boxes have a button to perform the listed action. some boxes have additional inputs such as dropdown menus and text fields.
